Build an effective website

Websites with optimized web pages can often attract the attention of consumers. So the project is to build a website, in order to attract more people to browse our page.

Value

Core values underpinning our project are:

  • quality
  • Respect customer needs.
  • Responsible

Methodology

We will use the waterfall model to design and develop the web page. First understand the customer's needs, design a draft web page according to the customer's needs, and then give the draft to the customer, and judge whether the expected goal is achieved according to the customer's feedback. The next step is to revise the draft based on the feedback. Then start making web pages and deliver them to customers. And continue to pay attention to the follow-up, maintain the website.
